Zadanie 1: Przejście lochu

Wejdź na stronę: CodeCombat - Dungeon.
Po wejściu na stronę zmień język programowania na JavaScript.

Dodatkowe wyzwanie: zamiast używać while(true), użyj pętli for. Oto przykład: 

for (let i = 0; i < 10; i++) {
hero.moveRight();
hero.moveDown();
}

Ta petla wykona sie 10 razy. Zamien wartosc 10 na 3 aby wykonac polecenia 3 razy.

Spróbuj też pisać

hero.moveRight(2);

Zamiast

hero.moveRight();
hero.moveRight();

Zadanie dodatkow: Zakładanie konta na GitHub i publikowanie strony

Krok po kroku:

  1. Wejdź na GitHub i załóż konto.
  2. Po zalogowaniu, kliknij przycisk "„Create repository".
  3. Podaj nazwę repozytorium (np. moja-strona) i "Create repository" na dole strony.
  4. W nowym repozytorium kliknij start by „creating a new file ”.
  5. Utwórz plik index.html i wpisz podstawowy kod HTML:
    6. <!DOCTYPE html>
<html>
<head>
<title>Moja Strona</title>
</head>
<body>
<h1>Witaj na mojej stronie!</h1>
</body>
</html>
  6. Zapisz plik, klikając „Commit changes”.
  7. Przejdź do zakładki „Settings” -> „Pages”.
  8. W sekcji „GitHub Pages” wybierz „Deploy from branch”, ustaw gałąź „main” i kliknij „Save”.
  9. Po chwili Twoja strona będzie dostępna pod adresem https://twojanazwa.github.io/moja-strona.
